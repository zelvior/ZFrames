ZFrames Documentation

About

ZFrames is an advanced workspace utility designed for high-performance multitasking. Open unlimited side-by-side grid windows to monitor, browse, or develop across multiple endpoints simultaneously.

Features & Usage

  • Dynamic Grid: The workspace automatically adjusts scaling based on the number of active frames.
  • Maxed Out Iframes: Edge-to-edge rendering without borders for a seamless viewing experience.
  • Note on Permissions: Due to web security standards (X-Frame-Options / Content Security Policy), some major websites (like Google or GitHub) block themselves from being loaded inside an iframe. This is a browser-level security feature.
