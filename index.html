ZFrames Documentation

About

ZFrames is an advanced workspace utility designed for high-performance multitasking. Open unlimited side-by-side grid windows.

Features & Usage

  • Smart URL Parser: Automatically handles search queries and domain formatting.
  • Iframe Bypass: Uses targeted parameters to allow Google Search inside frames.
  • Dynamic Grid: Scales automatically based on active frames.
